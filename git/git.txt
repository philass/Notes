Initializing a Git Repository
    -git init    -- type this in the desired directory
    -git add desiredFile1 desiredFile2 .....
    -git commit -m "YAY FIRST COMMIT BOY"
Then go to github and create a new repositiory
    -git remote add origin https://github.com/philiplassen/Notes.git
    -git push -u origin master

----------------------------------------------

Uninitializing a Git Repository
   -rm -rf .git
